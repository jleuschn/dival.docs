Dataset extracted from the LIDC-IDRI dataset.

The selection of image files is determined by the accompanying json file. It was generated by the script create_file_list.py.

Each image is cropped to the centered rectangle of shape (362, 362). The values are clipped to [-1024, 3071] HU and optionally (by default) normalized into [0., 1.] by the formula normalized = (original + 1024) / 4095.

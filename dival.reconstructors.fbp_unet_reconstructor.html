dival.reconstructors.fbp_unet_reconstructor module¶

Provides the learned FBPUNetReconstructor for the application of CT.

dival.reconstructors.fbp_unet_reconstructor.generate_fbp_cache(dataset, part, filename, ray_trafo, size=None)[source]¶

Write filtered back-projections for a CT dataset part to file.

Parameters

  • dataset (Dataset) – CT dataset from which the observations are used.

  • part ({'train', 'validation', 'test'}) – The data part.

  • filename (str) – The filename to store the FBP cache at (ending .npy).

  • ray_trafo (odl.tomo.RayTransform) – Ray transform from which the FBP operator is constructed.

  • size (int, optional) – Number of samples to use from the dataset. By default, all samples are used.

class dival.reconstructors.fbp_unet_reconstructor.FBPUNetReconstructor(ray_trafo, scales=5, epochs=20, batch_size=64, num_data_loader_workers=8, use_cuda=True, show_pbar=True, fbp_impl='astra_cuda', **kwargs)[source]¶

Bases: dival.reconstructors.reconstructor.LearnedReconstructor

CT Reconstructor applying filtered back-projection followed by a postprocessing U-Net (cf. 1). The U-Net architecture is similar to the one in 2.

References

1

K. H. Jin, M. T. McCann, E. Froustey, et al., 2017, “Deep Convolutional Neural Network for Inverse Problems in Imaging”. IEEE Transactions on Image Processing. doi:10.1109/TIP.2017.2713099

2

V. Lempitsky, A. Vedaldi, and D. Ulyanov, 2018, “Deep Image Prior”. IEEE/CVF Conference on Computer Vision and Pattern Recognition. doi:10.1109/CVPR.2018.00984

__init__(ray_trafo, scales=5, epochs=20, batch_size=64, num_data_loader_workers=8, use_cuda=True, show_pbar=True, fbp_impl='astra_cuda', **kwargs)[source]¶
Parameters

  • ray_trafo (odl.tomo.RayTransform) – Ray transform from which the FBP operator is constructed.

  • scales (int) – Number of scales in the U-Net.

  • epochs (int) – Number of epochs to train.

  • batch_size (int) – Batch size.

  • num_data_loader_workers (int) – Number of parallel workers to use for loading data.

  • use_cuda (bool) – Whether to use cuda for the U-Net.

  • show_pbar (bool) – Whether to show tqdm progress bars during the epochs.

  • fbp_impl (str, optional) – The backend implementation passed to odl.tomo.RayTransform in case no ray_trafo is specified. Then dataset.get_ray_trafo(impl=fbp_impl) is used to get the ray transform and FBP operator.
